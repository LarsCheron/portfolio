Student

Hi, I’m Nicolas – a passionate developer with a love for building efficient and scalable solutions. My journey in programming began with Python, where I mastered its simplicity and power to create dynamic applications, data pipelines, and automation tools. Driven by a need for performance and low-level control, I expanded my expertise to Rust, leveraging its speed and memory safety for robust system-level and backend projects. In addition to my expertise in Python and Rust, I have a deep appreciation for the web. From crafting seamless, user-friendly websites to developing full-stack applications, I enjoy combining clean design with robust functionality. My web development stack includes modern tools and frameworks like React, Flask, FastAPI, and Tailwind CSS, ensuring every project I create is not only functional but also visually appealing.
